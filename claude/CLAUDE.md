# Personal Claude Settings

## Style
- Be concise and direct — no filler, no preamble
- No emojis
- No trailing summaries after completing a task

## Code
- Minimal — no over-engineering, no unnecessary abstractions
- No comments unless logic is non-obvious
- No docstrings or type annotations on unchanged code
- Prefer editing existing files over creating new ones
- Delete unused code outright — no backwards-compat hacks

## Environment
- Shell: zsh
- Terminal: Ghostty
- Editor: Neovim
- Prompt: Starship
- OS: macOS
- Package manager: Homebrew
- Languages: Go, Python, JavaScript/TypeScript

## Dotfiles
- Stored at ~/Projects/dotfiles
